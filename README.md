# Sheenhaus — Premium Digital Studio

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → Import from GitHub
3. Deploy (zero config needed)
4. Add custom domain: sheenhaus.com

## Before Going Live — Replace These

- `Partner Name` → Your PM's actual name (in `app/page.tsx`)
- `91XXXXXXXXXX` → Your WhatsApp number (in `app/page.tsx`)
- `cal.com/sheenhaus/intro` → Your actual Cal.com booking link (in `app/page.tsx` + `components/Navbar.tsx`)

## Tech Stack

- Next.js 16 + TypeScript
- Tailwind CSS
- Framer Motion (animations)
- Deployed on Vercel (free)
